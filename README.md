## Overview

This project uses Machine Learning techniques to predict the selling price of a car based on various features such as car age, present price, kilometers driven, fuel type, seller type, transmission type, and ownership history.

The model is trained on historical car sales data and can estimate the market value of a car with good accuracy.

## Features

- Predict car selling prices
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Machine Learning model training and evaluation
- Easy-to-use Python implementation

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
